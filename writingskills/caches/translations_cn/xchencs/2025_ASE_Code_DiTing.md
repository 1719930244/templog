# Code-DiTing: Automatic Evaluation of Code Generation without References or Test Cases
# Code-DiTing：无需参考代码或测试用例的代码生成自动评估

**来源**: ASE 2025 | **作者**: Xiang Chen 等

## 摘要
代码生成的评估通常依赖参考代码或测试用例，但在实际场景中这些资源往往不可用。本文提出Code-DiTing，一种无需参考代码和测试用例的代码生成自动评估方法。通过分析生成代码的多维质量特征（语法正确性、语义一致性、代码风格等），实现自动化评估。

## 方法概述
1. 多维质量特征提取（语法、语义、风格、复杂度等）
2. 基于预训练模型的语义一致性评估
3. 综合评分模型融合多维特征

## 关键发现
- 无参考评估方法与人工评估的相关性达到较高水平
- 多维特征融合比单一维度评估更准确
- 在多个代码生成基准上验证了方法的有效性
