# Automated Query Reformulation for Efficient Search Based on Query Logs from Stack Overflow
# 基于Stack Overflow查询日志的自动查询重构以实现高效搜索

**来源**: ICSE 2021 | **作者**: Xiang Chen 等

## 摘要
开发者在Stack Overflow上搜索技术问题时，初始查询往往不够精确，导致搜索效率低下。本文提出基于查询日志的自动查询重构方法，通过分析SO上的历史查询模式，自动优化开发者的搜索查询。方法利用查询日志中的成功搜索模式来指导查询扩展和修改。

## 方法概述
1. 从SO查询日志中挖掘查询重构模式
2. 基于查询相似度匹配历史成功查询
3. 自动生成重构后的查询建议
4. 在真实搜索场景中评估重构效果

## 关键发现
- 自动重构的查询比原始查询的搜索效果显著提升
- 基于日志的方法比基于词典的传统方法更有效
- 查询重构对长尾查询的改善尤为明显
