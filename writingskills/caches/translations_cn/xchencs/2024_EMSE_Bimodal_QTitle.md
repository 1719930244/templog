# Automatic Bi-modal Question Title Generation for Stack Overflow with Prompt Learning
# 基于Prompt学习的Stack Overflow双模态问题标题自动生成

**来源**: EMSE 2024 | **作者**: Xiang Chen 等

## 摘要
Stack Overflow上的问题标题质量直接影响问题的可见性和回答率。许多开发者撰写的标题不够清晰或信息不足。本文提出一种基于Prompt学习的双模态（代码+文本）问题标题自动生成方法。通过同时利用问题描述文本和代码片段，生成更准确、更具信息量的标题。

## 引言核心
- SO上约30%的问题标题质量不佳，影响社区效率
- 现有标题生成方法仅使用文本信息，忽略了代码片段中的关键信息
- 代码和文本是互补的：代码提供技术细节，文本提供上下文
- 核心方法：Prompt学习框架融合代码和文本的双模态信息

## 方法概述
1. **双模态编码**：分别对问题描述文本和代码片段进行编码
2. **Prompt模板设计**：设计融合双模态信息的prompt模板
3. **标题生成**：基于预训练语言模型生成问题标题
4. **后处理**：去重、长度控制、格式规范化

## 关键实验发现
- 双模态方法比单模态（仅文本或仅代码）效果显著更好
- Prompt学习比传统微调在小数据场景下优势明显
- 生成的标题在人工评估中获得较高的相关性和流畅性评分

## 结论
双模态Prompt学习方法有效提升了SO问题标题的自动生成质量，为开发者社区的内容质量改善提供了技术支持。
